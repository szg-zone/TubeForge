# Person 1: Main Application & Configuration

## Files Worked On

### 1. TubeForgeApplication.java
- **Location:** `src/main/java/com/tubeforge/TubeForgeApplication.java`
- **What it does:** This is the main entry point of the Spring Boot application
- **Role:** Starts the entire application using `SpringApplication.run()`

### 2. CorsConfig.java
- **Location:** `src/main/java/com/tubeforge/config/CorsConfig.java`
- **What it does:** Configures Cross-Origin Resource Sharing (CORS)
- **Role:** Allows the frontend (running on localhost:3000) to make API calls to the backend (running on localhost:8080)

## Summary
Person 1 was responsible for:
- Setting up the Spring Boot application
- Configuring CORS to enable frontend-backend communication
- Creating the foundation that allows the app to run
